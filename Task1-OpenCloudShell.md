# Task 1: Open Cloud Shell

## Overview
In this task, you will sign in to the Azure Portal and open Cloud Shell — the browser-based terminal you will use to deploy resources.

---

## Steps

1. Open a browser and go to [https://portal.azure.com](https://portal.azure.com)

2. Sign in with the following credentials:
   - **Username:** <inject key="AzureAdUserEmail" enableCopy="true" style="color:blue"></inject>
   - **Password:** <inject key="AzureAdUserPassword" enableCopy="true" style="color:blue"></inject>

3. Click **No** if asked to stay signed in. Click **Cancel** on the Welcome popup.

4. In the Azure Portal, click the **Cloud Shell icon (>_)** in the top toolbar.

   > **Note:** If prompted to create storage for Cloud Shell, click **Create storage**. This takes about **2 minutes**.

5. Make sure the shell is set to **PowerShell** mode using the dropdown at the top-left of the shell panel.

---

## Summary

You have successfully:
- ✅ Signed in to the Azure Portal
- ✅ Opened Cloud Shell in PowerShell mode

Click **Next** to proceed to Task 2.
